# 🚛 Cálculo de Frete Mínimo

Aplicação desenvolvida em **Python** com **Streamlit** para calcular uma estimativa de frete mínimo com base em informações como origem, destino, quilometragem, categoria do veículo, ICMS, margem da empresa, peso da carga e retorno vazio.

---

## 📌 Funcionalidades

- Cadastro da cidade de origem e destino
- Seleção da categoria do veículo
- Informar quilometragem rodada
- Informar quantidade de eixos
- Opção para calcular retorno vazio
- Aplicação de margem da empresa
- Aplicação de percentual de ICMS
- Informar peso da carga em toneladas
- Seleção do tipo de carga
- Cálculo automático do frete estimado

---

## 🛠️ Tecnologias utilizadas

- Python
- Streamlit
